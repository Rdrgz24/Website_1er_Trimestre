How to create a paragraph

To create a paragraph we have to use the <p> tag

For example: <p>Paragraph example on HTML code</p>

How to create a heading "h1"

To create a heading level 1 we have to use te <h1> tag

For example: <h1>Heading example on HTML code</h1>

How to create a list an ordered list

To create a ordered list we have to use the <ol> tag and <li> tag

For example:
<ol>
<li>Elemento 1</li>
<li>Elemento 2</li>
<li>Elemento 3</li>
</ol>

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

How to create a link to a website

To link a website we have to use <a href="">Content</a> tag

For example: <a href="https://www.w3schools.com/">W3Schools> </a>

sample:

W3Schools

How to insert an image in a website

To insert an image we have to use <img src="" alt=""> tag, src is used to link the image and alt is used if the imagen is not in screen

For example: <img src="example.jpg" alt="img_example" style="width: 200px">
