CSS - Essentials

How to change the color to an h1 heading

To change the color we have to use the <style="color"> tag

For example: <h1 style="color:red">Heading h1 example</h1>

How to change the font family to an h1 heading

To change the font family we have to use the <style="font-family"> tag

For example: <h1 style="font-family: Calibri">Heading h1 example</h1>

sample:

Heading h1 example


How to change the background to an h1 heading

To change the background color and border we have to use <style="background-color: ; border: ;"> tag

For example: <h1 style="background-color: aqua; border: 1px solid red;"> Heading h1 example</h1>

How to align the text to the center

To align the text we have to use <h1 style="text-align"> tag

For example: <h1 style="text-align: center;"> Heading h1 example</h1>

How to change the margin of an heading

To change the margin of the text we have to use <h1 style="margin"> tag

For example: <h1 margin-top: 25px; margin-left: 25px;>Heading h1 example</h1>
