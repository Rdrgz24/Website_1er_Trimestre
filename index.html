Introduction to Languages

HTML (Hyper Text Markup Language)
Is the standard markup language for documents designed to be displayed in a web browser.

CSS (Cascading Style Sheets)
Is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML, including SVG, MathML or XHTML.

SVG (Scalable Vector Graphics)
Is an XML-based vector image format for defining two-dimensional graphics, having support for interactivity and animation
