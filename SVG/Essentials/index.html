How to create a circle in SVG

To create a circle in svg we have to use <svg> and <circle> tag

For example:
<svg width="100" height="100" xmlns="http://www.w3.org/2000/svg">
<circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow"/>
</svg>

How to create a Drop Shadow

To create a Drop Shadow in svg we have to use <svg> , <defs> , <filter> and <rect>

For example:
<svg height="110" width="110" xmlns="http://www.w3.org/2000/svg">
<defs>
<filter id="f1"> <rect>
<feDropShadow dx="12" dy="14" stdDeviation="1" flood-opacity="0.7"/>
</filter>
</defs>
<rect width="90" height="90" fill="yellow" filter="url(#f1)"/>
</svg>
